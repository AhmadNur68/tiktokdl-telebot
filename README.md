# tiktokdl-telebot
Simple TikTok Downloader Telegram Bot by [CaliphDev](https://caliph.my.id)

## Tutorial
### installation
```sh
git clone https://github.com/caliphdev/tiktokdl-telebot
npm install
```

### Edit config.js
Change Admin ID & Bot token
### 
```sh
nano config.js
```

### Running the scripts
```sh
node .
```
